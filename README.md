# Projeto CloneSpotify

Este é um projeto de clone do Spotify, desenvolvido para fins educacional. O objetivo é criar uma interface semelhante à do Spotify, utilizando tecnolgia web.


## Tecnologias Utilizadas 

- HTML5
- CSS3
- Typescript
- Tailwind CSS ( versão 4.x)
- Node.js ( versão 24.x)

## Instalação

1.  Clone este repositório:

``` bash
git clone https://github.com/marcellogomes-santos/CloneSpotify.git

```

2. Navegue até o diretório do projeto:

```bash
cd CloneSpotify

```

3. Instale as dependências:

```bash

npm install

```
